# Shared Data

Platform-agnostic seed data and shared resources used by both iOS and Android.

> **Note:** The iOS seed files (`exercises-seed.json`, `programs-seed.json`) currently live inside
> `ios/Packages/GymBroCore/Sources/GymBroCore/Resources/` because they are referenced by Swift
> bundle-loading code. They will be migrated here once the shared data loading layer is implemented.
